## `Methods`

### sheetLocation()

```js
sprite:sheetLocation() -> dict
```

Returns the x, y position of a sprite on the sprite sheet

---

### draw()

```js
sprite:draw() -> none
```

Draws the sprite on the canvas according to the properties
